What?

An intelligent visual analytic system that provides insight into the
Yelp Phoenix dataset

Why?

  • The reason behind the user provided ratings is unknown

  • Understand more about the restaurants in Phoenix with respect to
    food , price , service and ambience

  • Reduce the user's effort to choose a restaurant

  • Visualize the reviews effectively so that users can save time from reading all reviews

    How?

  • Text Mining of Yelp Reviews

  • Creating Effective Visualizations

    • Mining of Yelp Reviews


    Data Collection

  • From the Yelp Academic Dataset Challenge


    • Data Preprocessing of Reviews

  • Filter out the dataset for Phoenix restaurants

  • Tokenizing

  • Stop words and Punctuation Removal

    • Feature Selection

  • Extracting unigram,bigram and trigram features from the preprocessed data
    Reference: http://www.ics.uci.edu/~vpsaini/files/train.arff

  • Referred to the annotated feature set for each attribute :
    Food ,Price,Service and Ambience

  • Form the feature vectors for all the reviews


    • Model

  • Developed binary classifiers for each category

  • Used Random Forest Binary Classifier

  • Performed k - fold cross validation to improve the model accuracy
